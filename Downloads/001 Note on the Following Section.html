The following section in this course shows how to work with an older version of React Native.

Why is this content still here?

Turns out that many companies got started with React Native years ago, and use an earlier version of it.  Many of these companies have decided to continue using those older versions, rather than spend the time and resources to update.  Either way, they still need engineers who understand these older versions!  That's what these videos are here for: helping these companies train employees on how to work with legacy versions of React Native.

Do I need to watch these videos?

You only need to watch the videos in the next section if you know that you will be working on a project that involves an older version of React Native.  Otherwise, they can be skipped.
